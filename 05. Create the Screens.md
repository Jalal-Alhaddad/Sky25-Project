- Create Screens files in screens folder for `Home`, `Help`, `Not found`, `New & Edit`, `View`, and `Staff Directory` (6 screens)

```bash
|- screens

    |- HelpScreen.js
    |- NotFoundScreen.js
    |- PeopleViewScreen.js
    |- PersonViewScreen.js

```

- Write the basic screens layout
